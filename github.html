What is Git / GitHub ?

Git is a version control tool that

helps software engineers backup their code in an incremental fashion

allows multiple programmers to write code collaboratively and asynchronously

organizes code for a project as a repository

GitHub is

a web host of Git repositories

an online community of programmers

How to code collaboratively on GitHub?

All members of a team share a remote repository hosted on GitHub. Each member also has a local copy of the repository in their own computer. Members fetch from the remote repository to get other team member's work. When a member makes an update and is ready to share it with the other programmers, he pushes his update to the remote repository for others to fetch.

Three Key Concepts About Git

Commit
a snapshot of the current code
Branch
a series of commits
Merge
join two branches into one
